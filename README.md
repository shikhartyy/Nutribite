# NutriBite AI

NutriBite AI is a Flutter-based smart meal ordering and nutrition app focused on personalized healthy meals.

## Features

* Premium multi-screen Flutter UI
* Smart Meal Builder
* Add to Cart flow
* Meal Details page
* Responsive design
* Cross-platform support

## Tech Stack

* Flutter
* Dart
* Firebase (planned)
* REST API (planned)

## Status

Currently in progress with continuous feature and UI upgrades.
